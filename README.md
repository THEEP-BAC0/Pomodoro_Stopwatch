# Pomodoro Stopwatch
### Built with Python and PySide6

Pomodoro Stopwatch is a focus timer that adapts to how long you can actually concentrate, instead of forcing fixed time blocks.
It follows the core ideas of the Pomodoro Method while dynamically adjusting work sessions based on your behavior.

---

## Key Features

- Dynamic work sessions based on your real focus duration
- Minimal, distraction-free interface
- Manual control to start, stop, and reset sessions
- Lightweight desktop app built with PySide6
